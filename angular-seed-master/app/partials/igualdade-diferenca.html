Igualdade/Diferença

O Javascript possui dois operadores de igualdade/diferença.

Os operadores === e o !== são os operadores estritos. Ao usar esses operadores, os objetos só são considerados iguais caso eles sejam do mesmo tipo e possuam exatamente o mesmo valor.

            var bool1 = "1" === "1"; // => true
            var bool2 = 1 !== "1"; // => false

Os operadores == e != são usadas da mesma forma que os operadores estritos com a diferença que caso os valores não sejam do mesmo tipo, esses operadores admitem que sejam efetuadas conversões implicitas para que os valores possam ser considerados iguais. 

            var bool1 = null == undefined;
            var bool2 = 1 == "1";

Abaixo seguem exemplos do comportamento dos dois tipos de operadores. 

            1 == 1 // => true
            1 === 1 // => true
            "1" == "1" // => true
            "1" === "1" // => true
            1 == "1" // => true
            1 === "1" // => false
            false == false // => true
            false === false // => true
            null == undefined // => true
            null === undefined // => false
            1 == true // => true
            1 === true // => false
            "1" == true // => true
            "1" === true // => false
