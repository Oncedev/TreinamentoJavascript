Expressões Regulares

Baseada nas expressões regulares da linguagem Perl.

Define um padrão de carácteres que podem ser utilizadas para validar strings, fazer buscas em strings, fazer substituição de caracteres entre outros usos.

            var expr1 = /\d{2,4}/;
            var expr2 = /\w{3}\d?/;

A tabela abaixo mostra quais são os caracteres que podem ser usados em uma expressão regular.

Caracteres Padrão de match
Qualquer literal Faz o match dele mesmo
[...] Qualquer caracter dentros dos cochetes
[^...] Qualquer caracter exceto o dos cochetes
\w Qualquer caracter do ascii. [a-zA-Z0-9_]
\W Qualquer caracter não ascii
\s Qualquer espaço em branco
\S Qualquer caracter que não seja espaço em branco
\d Qualquer dígito
\D Qualquer caracter que não seja número

A medida que as expressões regulares se tornam mais complexas se torna necessário especificar a quantidade de repetições que um determinado conjunto de caracteres deve aparecer. Os caracteres abaixo são utilizados para o número de repetições.

Caracteres Significado
{n,m} O Item anterior deve aparecer pelo menos n vezes e não mais que m vezes
{n,} Item anterior deve aparecer pelo menos n vezes
{n} Item anterior deve aparecer exatamente n vezes
? Item anterior deve aparecer nenhuma ou uma vez. Equivalente a {0,1}
+ Item anterior deve aparecer pelo menos uma vez. Equivalente a {1,}
* Item anterior deve aparecer qualquer quantidades de vezes, incluindo zero. Equivalente a {0,}

Por último, existem as flags que especificam determinados comportamentos dos padrões de matching

Caracteres Significado
i Executa um match não case sensitivo
g Faz o match para todas as ocorrências ao invés de parar na primeira ocorrência

As expressões regulares podem ser executadas a partir do tipo String ou usando o objeto RegExp. Abaixo seguem exemplos nas duas maneiras.

            // Usando o tipo String
            "JavaScript".search(/script/i); // => 4: Retorna o indice da palavra buscada 
            
            "Javascript".replace(/javascript/gi, "JavaScript"); 
            /* => JavaScript: Faz replace na string sem 
             * considerar case sensitive e valendo para um número infinito de matches.
             */

            "1 plus 2 equals 3".match(/\d+/g); 
            // => ["1","2","3"]: Retorna os matchs para a expressão regular especificada.  

            // Usando o objeto RegExp
            new RegExp("\\d{5}", "g").exec("41500"); 
            // => ["41500"]: Retorna a string que corresponde ao padrão
            
            new RegExp("\\d{5}", "g").test("41500");
            // => true: Testa se a string corresponde ao padrão
