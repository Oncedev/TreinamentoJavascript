Objetos

Objetos são a unidade básica do javascript, com exceção dos números e strings todo o resto é considerado objeto.

Objetos funcionam como HashMaps ou Dictionary, onde chaves são strings e o valor são primitivos ou outros objetos.

Objetos podem ser criados de 3 formas diferentes.

Usando literais.

            var vazio= {};
            var ponto = { x:0, y:0 };
            var livro = {
                titulo: "JavaScript", 
                author: { 
                    nome: “Paulo”
                }
            };

Usando prefixo new.

            var o = new Object();
            var a = new Array();
            var d = new Date();
            var r = new RegExp("js");

Ou usando o método Object.create

            var Pessoa = {
                nome: "Paulo"
            };

            var paulo = Object.create(Pessoa);
            console.debug(paulo.nome);

As propriedades ou atributos de um objeto podem ser acessadas de duas maneiras, isso é bom porque nos traz flexibilidade quando estamos desenvolvendo nossas aplicações.

Podemos usar a dot notation para acessar as propriedades, como no exemplo abaixo.

            var pessoa = {
                nome: "Paulo",
                nacionalidade: "Brasil",
                idade: 25
            };

            pessoa.nome // => "Paulo"
            pessoa.nacionalidade // => "Brasil"
            pessoa.idade // => 25

Podemos usar uma notação semelhante aos hash maps/dictionary. Esse tipo de notação é muito útil quando não temos conhecimento do nome da propriedade antes de executar alguma operação.

            var pessoa = {
                nome: "Paulo",
                nacionalidade: "Brasil",
                idade: 25
            };

            pessoa["nome"] // => "Paulo"
            pessoa["nacionalidade"] // => "Brasil"
            pessoa["idade"] // => 25

Podemos testar a existência de uma propriedade em um determinado objeto de duas maneiras.

Usando o operador in.

            var pessoa = {
                nome: "Paulo",
                nacionalidade: "Brasil",
                idade: 25
            };

            "nome" in pessoa // => true
            "endereco" in pessoa // => false

Podemos testar a existência de uma propriedade em um determinado objeto de duas maneiras.

Usando o operador in.

            var pessoa = {
                nome: "Paulo",
                nacionalidade: "Brasil",
                idade: 25
            };

            "nome" in pessoa // => true
            "endereco" in pessoa // => false

Ou testando o valor da propriedade.

            var pessoa = {
                nome: "Paulo",
                nacionalidade: "Brasil",
                idade: 25
            };

            pessoa["nome"] // => "Paulo"
            pessoa["endereco"] // => undefined

Podemos, também, enumerar propriedades de objeto através de loops, como por exemplo, para copiar um objeto. 

            var pessoa = {
                nome: "Paulo",
                nacionalidade: "Brasil",
                idade: 25
            };

            var trabalhador = {
                salario: 760.0
            };

            trabalhador = extend(trabalhador, pessoa);

            trabalhador.nome // => "Paulo"
            trabalhador['nacionalidade'] // => "Brasil"
            trabalhador['salario'] // => 760.0
