Por que o Javascript está sendo tão usado ?

O surgimento do Ajax (Asynchronous Javascript and XML), que é uso de tecnologias como Javascript e XML para tornar as páginas web mais interativas através de solicitações de conteúdo assíncronas, colocou as páginas web em um outro nível de exigência. Desenvolvedores e designers passaram a ser preocupar muito mais com a experiência do usuário (UX - User Experience) e o Javascript assumiu um papel fundamental no desenvolvimento web.

Um conceito que tem ganhado bastante atenção recentemente é o SPA, ou Single Page Application, uma SPA é uma página web onde todas as interações do usuário são feitas na mesma página, a página não é recarregada em nenhum momento do processo. Novas informações são carregadas através de requisições assícronas entre a página e o servidor web.

Um exemplo de SPA é essa página, que usa um framework chamado AngularJS para gerenciar as transições entre as diferentes seções do tutorial. Outros exemplos são o GMail, Facebook e o Twitter.

Um outro fator que também tem influenciado o aumento da popularidade do Javascript é o node.js. O node.js é uma plataforma construída em cima da engine de Javascript V8, a mesma utilizada no Chrome, facilitando a escrita de aplicações real-time.

O Node.js faz com que possamos usar o Javascript também no lado servidor, eliminando a necessidade da equipe trabalhar com múltiplas linguagens de programação (normalmente as equipes usam o Javascript no front-end e algumas outra linguagem de programação no back-end, como por exemplo, ruby, python ou C#).

Algumas empresas grandes como o Linkedin e o Walmart, estão reportando ganhos consideráveis de performance e produtividade ao começarem a usar Node.js para suas aplicações.
