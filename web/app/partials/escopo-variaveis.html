Escopo de variáveis

No Javascript as variáveis podem possuir escopo global, quando declaradas no objeto global, ou escopo local quando declaradas dentro de objetos ou de funções.

Variáveis declaradas no escopo global são sobreescritas por variáveis escopo local, caso ambas possuam o mesmo nome.

No Javascript não possuem variáveis de escopo de bloco, variáveis declaradas dentro de blocos passam automaticamente a fazer parte do escopo local

Confira alguns exemplo do uso de escopos abaixo.

            var global = "escopo global";

            function imprimeVariavelGlobal() {
                console.debug(global); // => escopo global
            }

            function imprimeVariavelLocal() {
                var global = "123"; 
                console.debug(global); // => 123, variável local sobreescreve variável global
            }

            function imprimeVariabelComEscopoDeBloco() {

                if (true) {
                    var escopoBloco1 = 123;     
                } else {
                    var escopoBloco2 = 456;     
                }

                console.debug(escopoBloco1); // => 123, variável declarada no bloco está no escopo local
                console.debug(escopoBloco2); // => undefined
            }
