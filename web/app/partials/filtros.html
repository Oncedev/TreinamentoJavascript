Filtros

Seletores também podem ser combinados com filtros na hora de consultar elementos no DOM.

Alguns exemplos de filtros que podem ser utilizados são:

O filtro :eq(n), é utilizado para obter o elemento cujo índice é n.

O filtro :even e :odd são utilizados para pegar apenas os elementos de índice par e ímpar respectivamente. Os índices começam com zero.

O filtro :not é utilizado para pegar os elementos apenas os elementos que não atendem a um determinado seletor.

            $(document).ready(function() {
                $("table tr:even td").each(function() {
                    $(this).css('color', 'blue');
                }); // Pinta a primeira e a terceira linha de azul

                $("table tr:odd td").each(function() {
                    $(this).css('color', 'blue');
                }); // Pinta a segunda e a quarta linha de azul

                $("table tr:not(:odd) td").each(function() {
                    $(this).css('color', 'blue');
                }); // Pinta a primeira e a terceira linha de azul

                $("table tr:eq(2) td").each(function() {
                    $(this).css('color', 'blue');
                }); // Pinta a terceira linha de azul
            });
