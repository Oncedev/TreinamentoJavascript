Seletores CSS

O uso de seletores CSS para fazer consultas ao DOM é umas das coisas mais legais no jQuery, em alguns casos, seu uso poupa o desenvolvedor de escrever dezenas de código javascript puro para atingir o mesmo resultado.

Os seletores CSS podem ser de 4 maneiras diferentes. Podemos consultar um elemento pelo seu id, por sua classe, por sua tag ou por algum atributo do elemento, ou se quisermos algo mais específico, também podemos combinar esses 4 tipos de seletores para formar novos seletores.

Consultando elementos pelo id

Podemos consultar elementos no DOM pelo id através do uso do carácter '#' + o id do elemento. Confira o exemplo abaixo.

            

                Minha div1
            

            

                Minha div2

No DOM acima, temos duas divs cada uma com seus respectivos id, agora vamos usar o jQuery para extrair o conteúdo interno de cada uma delas. 

            $(document).ready(function () {
                $("#div1").html(); // => Minha div1
                $("#div2").html(); // => Minha div2
            });

Consultando elementos pela classe

Uma outra forma de consultar elementos é pela classe. Para executar consultas no DOM pela classe dos elementos usando o caracter '.' + a classe que queremos buscar. Confira o exemplo abaixo. 

            

                LA Lakers
            

            

                Miami Heat
            

            

                Barcelona
            

            

                New England Patriots

Para pegar apenas os times de basquete, podemos usar o comando abaixo.

            $(document).ready(function () {
                $(".timeBasquete").each(function() {
                    console.debug($(this).html());    
                })
            });

Consultando elementos pela tag

Podemos também consultar os elementos pelas tags html. 

            

                Div1
            

            

                Div2
            

            

                P1
            

            

                P2

Podemos pegar apenas o conteúdo dos parágrafos

            $(document).ready(function () {
                $("p").each(function() {
                    console.debug($(this).html());    
                })
            });

Consultando elementos por atributos

Podemos também consultar os elementos pelos atributos.

Podemos pegar apenas o conteúdo dos campos de texto da seguinte forma.

            $(document).ready(function () {
                $("input[type='text']").each(function() {
                    console.debug($(this).val());    
                })
            });

Combinando seletores

Por fim, seletores também podem ser combinados para gerar consultas.

            

                
                    

                        1
                    

                    

                        2
                    

                    

                        3
                    

                    

                        4

O código abaixo usa seletores de tags, ids e classes para pintar os número impares de verde e os números pares de azul.

            $(document).ready(function () {
                $("table#tabela tr.impar").each(function() {
                    $(this).css('color', 'green');            
                });
                
                $("table#tabela tr.par").each(function() {
                    $(this).css('color', 'blue');            
                });
            });
