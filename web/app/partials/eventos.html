Eventos

Podemos programar javascript usando desenvolvimento guiado a eventos, e essa é a forma em que normalmente ele é aplicado. O desenvolvimento web, por exemplo, que é onde há o maior uso do javascript, nós costumamos escrever eventos para os nossos controles HTML que vão ser disparados a medida que os usuários navegam na página.

Eventos no javascript podem ser declarados no html, usando javascript puro ou usando as abstrações do jQuery. Como essa seção trata de jQuery, irei focar em como atribuir eventos aos elementos usando apenas o jQuery.

Eventos suportados pelo jQuery e consequentemente pelo javascript podem ser divididos em 5 categorias.

Eventos de formulário

Os elementos formulários, representados pela tag form, possuem 2 eventos, o evento submit que é quando o formulário é submetido e o evento reset que é disparado quando o formulário é resetado.

O código abaixo é usado para atribuir ambos os eventos, o jQuery apresenta atalhos para a definição de alguns métodos, o método submit é um deles. Para o método reset não existe atalhos, sendo necessário a atribuição do método através da método bind. 

            $(document).ready(function() {
                $("form").submit(function() {
                    alert("submit");
                });
                
                $('form').bind('reset',function(){
                    alert("reset");
                });
            });

Eventos de elementos de formulário

Elementos como textbox, select, radio button e etc também possuem seus eventos. Esse eventos estão normalmente associados a alteração de valor, ou a interações com os elementos.

O evento change é disparado quando um determinado elemento tem o seu valor alterado.

Os eventos focus e blur são disparados quando um elemento ganha ou perde o foco respectivamente.

O código abaixo mostra como podemos atribuir esses eventos.

            $(document).ready(function() {
                $("input").focus(function() {
                    $("form").append("
Focus
");
                });
                
                $("input").blur(function() {
                    $("form").append("
Blur
");
                });
                
                $("select").change(function() {
                    $("form").append("
Change
");
                });
            });

Para ver o código acima funcionando clique aqui.

Eventos associados a janela do browser

Além dos eventos focus e blur que também podem ser associados a janela do browser, outro evento pode ser associado a janela do browser que é o evento resize.

O evento resize como o próprio nome diz, é disparado quando a página é redimensionada.

O código abaixo mostra a aplicação do resize.

            $(document).ready(function() {
                $( window ).resize(function() {
                    $("div").append("resize");
                });
            });

Eventos associados ao mouse

Os eventos de mouse são normalmente disparados quando um usuário clica ou movimenta o mouse.

O evento mousemove é disparado toda vez que o usuário movimenta o mouse na nossa página.

Os eventos mousedown e o mouseup são disparados quando o usuário respectivamente pressiona o botão do mouse e quando o usuário libera o botão do mouse, respectivamente.

Os eventos mouseover e mouseout são utilizados, respectivamente, quando o usuário coloca o mouse sobre um elemento ou quando o usuário retira o cursor do mouse do elemento.

Por último, o evento click é disparado quando o usuário clica no mouse, enquanto o evento dblclick é disparado quando o usuário clica 2 vezes em um curto período de tempo.

Abaixo, segue o código que faz a atribuição desses 7 eventos.

            $(document).ready(function() {
                $(window).click(function() {
                    $("div").append(" click ");
                });
                
                $(window).dblclick(function() {
                    $("div").append(" dblclick ");
                });
                
                $(window).mousemove(function() {
                    $("div").append(" mouse move ");
                });
                
                $(window).mouseup(function() {
                    $("div").append(" mouse up ");
                });

                $(window).mousedown(function() {
                    $("div").append(" mouse down ");
                });

                $(window).mouseover(function() {
                    $("div").append(" mouse over ");
                });

                $(window).mouseout(function() {
                    $("div").append(" mouse out ");
                });                        
            });

Eventos associados ao teclado

Existem 3 eventos de teclado que podem ser executados pelo browser, o keydown, o keypress e o keydown.

O evento keydown e keyup são disparados quando o usuário, respectivamente, pressiona uma tecla do teclado e quando ele libera essa tecla. O evento keypress é liberado toda vez que um novo carácter é gerado pelo teclado, que acontece a cada vez que uma tecla é, ou permanece, pressionada.

O código abaixo mostra como podemos atribuir esses eventos. 

            $(document).ready(function() {
                $(window).keyup(function() {
                   $("div").append("key up"); 
                });
                
                $(window).keydown(function() {
                   $("div").append("key down"); 
                });
                
                $(window).keypress(function() {
                   $("div").append("key press"); 
                });    
            });
