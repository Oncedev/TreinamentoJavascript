Estrutura Léxica

O Javascript é uma linguagem case-sensitive. 

			paulo == paulo
			Paulo != paulo

Comentários no Javascript podem ser comentários de uma linha ou comentários de múltiplas linhas. 

			// Uma linha de comentário

			/*	Várias linhas
			 *	de comentário      
			 */

Comentários no Javascript podem ser comentários de uma linha ou comentários de múltiplas linhas. 

			12    // Número 12
			1.2  // Número decimal 1.2
			"hello world" // Uma string declarada com asdpas duplas
			'Oi' // String declarada com aspas simpels 
			true // Valor booleano
			false // Outro valor booleano
			/javascript/gi 	// Literal representando uma expressão regular
			null 			// Sem valor
			undefined 	// Valor não declarado
			{ x:1, y:2 }        // Objeto com atributos x e y
			[1,2,3,4,5]       // Array de valores

Ponto e virgula são opcionais, porém é uma boa prática usa-los, já que podem causar erros de interpretação. 

			function comPontoVirgula() {
				return true;
			}

			function semPontoVirgula() {
				return
					false;
			}

			console.debug(comPontoVirgula()); // => true
			console.debug(semPontoVirgula()); // => undefined
