Manipulando o DOM

Uma outra feature importante do jQuery é facilitar a manipulação do DOM, a medida que as aplicações ficam cada vez mais responsivas e interativas é necessário que seja adicionado, alterado ou removido elementos do DOM de acordo com as ações do usuário.

O jQuery oferece alguns métodos em sua API padrão para fazer a manipulação de elementos do DOM.

Método Descrição
append Insere o conteúdo ao final do elemento
prepend Insere o conteúdo no ínicio do elemento
after Insere o conteúdo antes do elemento
before Insere o conteúdo depois do elemento
empty Deleta o contéudo do elemento
remove Deleta o elemento

Abaixo segue alguns exemplos da utilização desses métodos.
