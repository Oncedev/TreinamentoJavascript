Arrays

Arrays são objetos utilizados para armazenar coleções.

Um array no Javascript pode ser declarado de duas formas diferentes.

Usando literais.

            var array = []; // Array vazio
            var array = [1,2,3]; // Criando um array já preenchido

Ou usando o prefixo new.

            var array = new Array();
            var array = new Array(1,2,3);

Posições do array são acessadas através da sintaxe [posição].

            var array = [1,2,3];
    
            array[0] = 5; // => [5,3,3]
            var pos = array[2]; // => 3

Alguns métodos já são implementados por default no array. Abaixo segue uma lista de métodos interessantes.

O método join é usado para concatenar todos os itens do array usando um separador pré-definido.

            var arrayJoin = [1,2,3];
            arrayJoin.join(); // => "1,2,3"
            arrayJoin.join("|"); // => "1|2|3"
            arrayJoin.join("-"); // => "1-2-3"

O método reverse é usado inverter a ordenação de um array.

            var arrayReverse = [1,2,3];
            arrayReverse.reverse();

O método sort é usado para ordenar um array, pode ser usado tanto com a ordenação natural para os elementos do array, como também usando funções customizadas.

            var arraySort = [33, 4, 1111, 222];
            arraySort.sort(); // => [1111, 222, 33, 4]
            arraySort.sort(function(a,b){return a-b;}); // => [4, 33, 222, 1111]
            arraySort.sort(function(a,b){return b-a;}); // => [1111, 222, 33, 4]

O método concat é usado para gerar um novo array concatenado com os novos elementos.

            var arrayConcat = [1,2,3];
            arrayConcat.concat([4,5]);

O método slice, como o próprio nome diz, é usado para extrair um corte do array original.

            var arraySlice = [1,2,3,4,5];

            arraySlice.slice(0,3); // => [1, 2, 3]
            arraySlice.slice(3); // => [4, 5]
            arraySlice.slice(1,-1); // => [2, 3, 4]
            arraySlice.slice(-3,-2); // => [3]

Diferente do método slice e do concat que geram um novo vetor do derivado do array original, o splice extrai um pedaço ou subarray do vetor original, porém modificando o vetor, removendo a parte retornada.

            var arraySplice = [1,2,3,4,5,6,7,8];

            arraySplice.splice(4); // => [5, 6, 7, 8]
            arraySplice.splice(1,2); // => [2, 3]
            arraySplice.splice(1,1); // => [4]

Os métodos push e pop permitem que trabalhemos com arrays como se fossem pilhas, empilhando e desempilhando elementos. 

            var stack = []; 

            stack.push(1,2); // => [1, 2]
            stack.pop(); // => [2]
            stack.push(3); [1, 3]
            stack.pop(); [3]
            stack.push([4,5]); [1, [4, 5]]
            stack.pop(); [4, 5]
            stack.pop(); [1]

O ECMAScript5 definiu 9 novos métodos para os arrays.

O método forEach recebe uma função como parâmetro e executa essa função para todos os elementos do array. Por exemplo, podemos usar esse método para somar todos os elementos do array.

            var array = [1, 2, 3, 4, 5];
            var soma = 0;
            array.forEach(function(x) {soma += x});

            soma // => 15

O método map recebe uma função e retorna um novo array onde cada elemento é o resultado da aplicação da função passada por parâmetro. Por exemplo, podemos usar esse método para duplicar todos os elementos de um array. 

            var array = [1, 2, 3, 4, 5];
            array.map(function(x) {return x * 2}); // => [2, 4, 6, 8, 10]

O método filter recebe uma função e retorna todos os elementos do array que atendem uma determinada condição. 

            var array = [1, 2, 3, 4, 5];
            array.filter(function(x) {return x > 2}); // => [3, 4, 5]

O método every recebe uma função e retorna true se a função retorna true para todos os elementos do array, do contrário, retorna false. 

            var arraySortidos = [1, 2, 3, 4, 5];
            var arrayPares = [2, 4, 6, 8, 10];
            var isPar = function(x) {return x%2 == 0};

            arraySortidos.every(isPar); // => false
            arrayPares.every(isPar); // => true

O método some recebe uma função e retorna true se a função retorna true para pelo menos um elemento do array, do contrário, retorna false. 

            var arraySortidos = [1, 2, 3, 4, 5];
            var arrayPares = [2, 4, 6, 8, 10];
            var isImpar = function(x) {return x%2 == 1};
 
            arraySortidos.some(isImpar); // => true
            arrayPares.some(isImpar); // => false

O método indexOf faz a busca de um elemento no array, retornando o índice do primeiro elemento encontrado que satisfaz a condição. 

            var array = [0, 1, 0, 1, 0];
            array.indexOf(1); // => 1

O método lastIndexOf faz a busca de um elemento no array, retornando o índice do último elemento encontrado que satisfaz a condição. 

            var array = [0, 1, 0, 1, 0];
            array.lastIndexOf(1); // => 3

O método reduce combina o resultado de dois elementos do array, retornando o resultado dessa operação para que seja combinado com os próximos elementos do array e assim seguidamente até que resulte em apenas um valor. O método reduceRight é usado da mesma forma que o reduce, a diferença é que no reduceRight as combinações iniciam no final do array e seguem até o seu início. 

            var array = [1,2,3,4,5]
            var soma = array.reduce(function(x,y) { return x+y }); // 15: Soma dos valores
            var produto = array.reduce(function(x,y) { return x*y }); // 120: Multiplicação dos elementos
            var maior = array.reduce(function(x,y) { return (x>y)?x:y; }); // 5: Maior valor
